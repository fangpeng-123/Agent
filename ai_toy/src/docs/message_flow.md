# 消息流文档

## 对话流程

```
用户: "合肥今天天气怎么样？"
    ↓
意图分类: tool_call (weather)
    ↓
工具执行: get_weather_now(city="合肥")
    ↓
工具结果: "合肥今天晴，气温15℃..."
    ↓
LLM 回复: "合肥今天天气晴朗，最高气温..."
```

## 消息结构

```json
{
  "role": "user",
  "content": "合肥今天天气怎么样？"
}
```

## 性能指标

| 阶段 | 典型耗时 |
|------|---------|
| 意图分类 | ~0.03 ms |
| 工具执行 | ~800 ms |
| LLM 回复 | ~1300 ms |
| 总耗时 | ~2-5 s |
